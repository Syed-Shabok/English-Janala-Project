English Janala is a dynamic educational platform designed to help users master English vocabulary efficiently. It bridges the gap between learning and practice by providing a categorized, interactive interface powered by real-time data.

---

## 📖 Project Overview
English Janala serves as a comprehensive tool for language learners to discover, hear, and understand new words. The application organizes vocabulary into specific levels, allowing users to progress at their own pace. By integrating a "Window" (Janala) to the English language, the project focuses on visual clarity, phonetic accuracy, and contextual understanding.

---

## 🛠 Main Technologies
This project is built using modern web standards to ensure speed and accessibility:
- **HTML5:** Semantic structure for high accessibility.
- **Tailwind CSS / CSS3:** Modern, responsive styling following Figma design principles.
- **JavaScript (ES6+):** Core logic for DOM manipulation, API integration, and state management.
- **Web Speech API:** Native browser integration for real-time English pronunciation.

---

## ✨ Main Features
- **Dynamic Level-Based Learning:** Lessons are fetched from APIs and displayed as interactive buttons.
- **Contextual Word Cards:** Each card displays the word, its meaning, and phonetic pronunciation.
- **Interactive Details Modal:** Provides deep-dive info, including example sentences and synonyms for every word.
- **Smart Search Functionality:** A real-time search bar that filters words across levels instantly.
- **Voice Synthesis:** A built-in audio feature allowing users to hear the correct native pronunciation.
- **State Management:** Highlights active lessons and resets filters during search for a seamless UX.

---

## 📦 Project Dependencies
To keep the application lightweight and fast, the project utilizes minimal external dependencies:
- **FontAwesome:** For intuitive UI icons (Sound, Info, etc.).
- **Google Fonts:** Utilizing high-readability fonts for educational content.
- **REST API:** For dynamic data fetching of lessons and vocabulary sets.
